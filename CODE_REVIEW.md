

1. Lead Management

Lead Source Dropdown

The Lead Form should include the following source options:

* WhatsApp
* Instagram
* Google
* Meta Ads
* Referral
* Walk-in
* Others

Lead Status Workflow

* Irrelevant leads should be managed separately.
* Once a lead is quoted and the status is changed to Appointment Confirmed, it should automatically move to the Kanban Board.
* When a lead is moved from Quoted to Appointment Confirmed, a new Appointment Form should automatically pop up to capture the remaining details.
* The appointment should then be saved in the Appointments Section.
* When the appointment status is marked as Completed, the lead should automatically move to Closed Won.

Data Export

* Enable data sheet export functionality from the Leads module.

3. Orders Management

Order Dates

* Trial Date should appear before Delivery Date in the order workflow.

Scheduling Calendar

* A scheduling calendar is required for:
    * Trial Date
    * Delivery Date
    * Delivery Time

Conflict Checking

* The system should automatically check for scheduling conflicts when selecting Delivery Date and Delivery Time.
* Users should be notified if another order is already scheduled for the selected slot.

Measurement Sheet

* The Orders module should use the same measurement sheet format as the House of Tailors measurement sheet.
* Measurements should be displayed in a structured grid format.as provided when print exported
